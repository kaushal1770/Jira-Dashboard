# RentEase – Furniture & Appliance Rental Platform (Agile Project Management)

## Project Overview

RentEase is an online Furniture & Appliance Rental Platform managed using Agile Scrum methodology. This repository showcases the complete project management lifecycle, including project planning, backlog management, sprint execution, stakeholder communication, risk management, and project documentation.

This repository focuses on the **project management process** rather than application source code.


# My Role

**Technical Project Manager**

I managed the project using Agile Scrum practices and Jira while coordinating project planning, sprint execution, backlog prioritization, and documentation.
